# Send SMS Using Python

------------------------------------------------------------------------------------
1)Open [fast2sms](https://www.fast2sms.com/) Website And Signup Using Your Mobile Number

-------------------------------------------------------------------------------------
2)In Dashboard Go To DEV-API

![](images/2.png)
-------------------------------------------------------------------------------------
3)Copy The API Key And Paste It In the Variable "api"

![](images/3.png)
-------------------------------------------------------------------------------------
